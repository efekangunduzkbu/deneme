# Deneme

Bu depo, ileride gelistirilecek basit bir oyun icin ornek frontend yapisini icerir. Su an icin kullaniciya mobil uyumlu bir ana sayfa, giris ve kayit sayfalari sunar.

## Dosyalar

- `index.html`: Kullanici karsilama sayfasi. Oyuna gecis icin butonlar ve giris/kayit baglantilari bulunur.
- `login.html`: Giris formu.
- `signup.html`: Kayit formu.
- `style.css`: Tum sayfalar icin basit ve mobil uyumlu stiller.

Sayfalari acmak icin bir web sunucusu kullanabilir veya dosyalari dogrudan tarayicinizda calistirabilirsiniz.
